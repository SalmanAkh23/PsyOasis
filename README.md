PsyOasis Web (Next.js)

Quick start:

1. cd web
2. npm install
3. npm run dev

This scaffold implements the basic structure for PsyOasis: landing page, Tailwind, Next.js (TypeScript). Add components and pages for directory, booking, auth, admin panel, and Firebase integration.

Environment (recommended)

- Use Node LTS. This repo includes `.nvmrc` set to `18.16.0`.
- To switch Node versions with nvm:

```bash
nvm install
nvm use
```

- Then install dependencies and run dev server:

```bash
cd web
npm install
npm run dev
```
